Encryption Tool

Standalone tool for encrypting parva content. Runs entirely in your browser — nothing is uploaded anywhere.

1. Set up encryption (one-time)

Pick a passphrase. A random salt is generated and a key is derived. Save the resulting encryption.json to parts/encryption.json in the repo.

2. Encrypt files

Load your existing encryption.json, enter the passphrase, then select content files to encrypt. Each file produces a <name>.enc download.
